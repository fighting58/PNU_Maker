karpathy-guidelines에 따라 작성해줘

1. 엑셀 또는 csv 파일 입력
2. 헤더 중에서 '행정구역'을 의미하는 헤더 선택
3. 헤더 중에서 '지번'을 의미하는 헤더 선택
4. 행정구역과 지번을 기준으로 PNU필드를 생성하고 엑셀에 추가
    1. sample-script.js에서 다음 기능을 가져와줘
       1. pnu 생성기능
       2. 행정코드 DB 다운로드 및 관리, 초기화 기능
5. 추가 기능
   1. 최종 생성 파일에서 모든 PNU 필드값들을 리스트 형태로 생성
   2. 클립보드로 복사 기능 추가
6. 모던한 UI로 구현

7. 파워셀에서 서버를 구동하고 포트 8000번으로 접속하면 웹페이지가 열리도록 구현하도록 bat 파일 생성
